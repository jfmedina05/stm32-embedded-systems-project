# STM32 Embedded Systems Project

A semester-long embedded systems repository built with STM32CubeIDE and STM32 microcontroller development tools. This project documents hands-on work in serial communication, GPIO control, sensor interfacing, and low-level embedded programming through a sequence of lab-based implementations.

## Overview
This repository captures my progression through a structured embedded systems course using STM32 microcontrollers. Across multiple labs, I worked with hardware-level programming concepts including UART communication, GPIO control, sensor integration, and peripheral configuration.

## Project Goals
- Build practical embedded systems experience using STM32 microcontrollers
- Develop comfort with hardware-software interaction at the microcontroller level
- Implement communication, control, and sensing features in C
- Strengthen embedded debugging and system design skills

## Tech Stack
- C
- STM32CubeIDE
- STM32 microcontrollers
- UART communication
- GPIO
- Embedded systems development

## Covered Topics
- GPIO configuration and output control
- UART communication and terminal interaction
- Command-based embedded interfaces
- Sensor-based input systems
- Peripheral initialization and microcontroller setup
- Real-time interaction between software and hardware
